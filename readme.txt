Instructions to use Clever Company Collage Creator

Prerequisites:
- Have opencv, numpy, and PILLOW (PIL fork) python libraries installed

(Install opencv)
pip install opencv-python

(Install numpy)
pip install numpy

(Install Pillow)
python3 -m pip install --upgrade pip
python3 -m pip install --upgrade Pillow

1. Drop a png image of your companies logo/image in the "images" folder and name it "originalPicture.png"
*Note: The image must have one color present that is NOT black or white.*

2. Run the file "image.py" and it will provide you with further instructions.

3. Output image will be in images folder and will be named finalimg.png

4. Enjoy!
